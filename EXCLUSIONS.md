This file records patent exclusions made pursuant to Section 2 (Patent License) of the [LICENSE.md](./docs/4_License.md) file of this repository which is based on the Community Specification License 1.0.

## Purpose
Under the Community Specification License 1.0, contributors who participate in the development
of this specification grant a patent license covering patents they own or control that are
necessarily infringed by a normative implementation of the specification.
A contributor may exclude specific patents or patent applications from this grant by submitting
a written exclusion notice within the exclusion period defined by the Working Group for each
approved draft. Exclusions must identify the specific patent or application being excluded and
the normative section(s) of the specification to which the exclusion applies.
Exclusions filed in accordance with this process are recorded in this file.

## Filing an Exclusion
To file a patent exclusion:

1. Open a pull request adding your exclusion to the Recorded Exclusions table below
2. The pull request must be submitted within the exclusion period for the applicable draft
3. Include the following information:

    - Contributor: Legal name of the contributing organization or individual
    - Patent / Application: Patent number(s) or application number(s) being excluded
    - Specification Section(s): The normative section(s) of the specification to which the exclusion applies
    - Applicable Draft: The Working Group draft version against which the exclusion is filed
    - Date Filed: Date the pull request was opened



**Exclusions must be filed against a specific draft. An exclusion filed after the exclusion period *(45 days)*
for a given draft has closed will not be effective for that draft.**

For questions about the exclusion process, contact the Working Group chair or open a GitHub issue.
